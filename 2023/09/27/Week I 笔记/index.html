概念

Computer terminal 终端

  • 硬件，数据输入输出的媒介（键盘、屏幕）
    • 历史：打字机，纸带
  • terminal emulator software

Console 控制台

今天可以认为控制台和终端没什么区别

![[Pasted image 20230925105522.png]]

Shell

读取并解释命令的程序，运行在terminal中，与kernel交互
  • explorer.exe
    • 文件资源管理器
  • cmd.exe
  • powershell（比CMD更强大）
  • bash
